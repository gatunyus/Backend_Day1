# 00 · เตรียมตัว: XAMPP & phpMyAdmin

## จุดประสงค์
เปิดเครื่องมือให้พร้อม และสร้างฐานข้อมูลแรกของเรา เพื่อใช้ฝึกตลอด Day 1

---

## 1. XAMPP คืออะไร?
XAMPP คือชุดโปรแกรมที่รวมทุกอย่างที่ต้องใช้ทำเว็บ Backend ไว้ในที่เดียว:

| ส่วนประกอบ | หน้าที่ |
| :-- | :-- |
| **Apache** | เว็บเซิร์ฟเวอร์ (รันไฟล์ `.php`) |
| **MySQL / MariaDB** | ฐานข้อมูล (เก็บข้อมูล) |
| **PHP** | ภาษาเขียนโปรแกรมฝั่งเซิร์ฟเวอร์ |
| **phpMyAdmin** | หน้าจอเว็บไว้จัดการฐานข้อมูลด้วยเมาส์ |

> Day 1 เราจะใช้แค่ **MySQL** กับ **phpMyAdmin** ส่วน Apache/PHP จะได้ใช้เต็มๆ ใน Day 2

---

## 2. เปิดใช้งาน
1. เปิดโปรแกรม **XAMPP Control Panel**
2. กด **Start** ที่ **Apache** และ **MySQL** ให้ขึ้นแถบสีเขียว

```
XAMPP Control Panel
┌─────────────────────────────────────────────┐
│ Module    │ PID    │ Port      │ Actions    │
│ Apache    │ 1234   │ 80, 443   │ [Stop] (on)│
│ MySQL     │ 5678   │ 3306      │ [Stop] (on)│
└─────────────────────────────────────────────┘
   ↑ ทั้งสองแถวเป็นสีเขียว = พร้อมใช้งาน
```

> ถ้า Start ไม่ขึ้น (มักเป็น Apache ติด port 80) ให้ปิดโปรแกรมที่แย่ง port เช่น Skype หรือเปลี่ยน port ของ Apache เป็น 8080

## 3. เข้า phpMyAdmin
เปิดเบราว์เซอร์ พิมพ์:

```
http://localhost/phpmyadmin
```

จะเจอหน้าจอจัดการฐานข้อมูล แถบซ้ายคือรายชื่อ Database ตรงกลางคือพื้นที่ทำงาน

แท็บสำคัญที่จะใช้บ่อย:

| แท็บ | ใช้ทำอะไร |
| :-- | :-- |
| **Databases** | ดู/สร้างฐานข้อมูล |
| **SQL** | พิมพ์คำสั่ง SQL เอง (เราจะใช้แท็บนี้มากที่สุด) |
| **Structure** | ดูโครงสร้างตาราง (คอลัมน์, ชนิดข้อมูล) |
| **Browse** | ดูข้อมูลในตารางแบบตาราง Excel |
| **Import / Export** | นำไฟล์ `.sql` เข้า / สำรองออก |

## 4. สร้างฐานข้อมูลแรก
เราจะสร้างฐานข้อมูลชื่อ `atsoft_day1` ไว้ฝึกวันนี้

**วิธี A — ใช้เมาส์ (GUI):**
1. คลิกแท็บ **Databases**
2. ช่อง "Database name" พิมพ์ `atsoft_day1`
3. ช่อง Collation เลือก **`utf8mb4_general_ci`** (รองรับภาษาไทย/อิโมจิ)
4. กด **Create**

**วิธี B — ใช้คำสั่ง SQL (แนะนำให้ลองด้วย):**
คลิกแท็บ **SQL** แล้ววางคำสั่งนี้ กด **Go**

```sql
CREATE DATABASE atsoft_day1
  CHARACTER SET utf8mb4
  COLLATE utf8mb4_general_ci;
```

> **ทำไมต้อง utf8mb4?** เพราะรองรับภาษาไทยและอิโมจิได้ครบ ถ้าเลือกผิดอาจเจอตัวอักษรเพี้ยนเป็น `???`

เมื่อสร้างเสร็จ จะเห็น `atsoft_day1` โผล่ในแถบซ้าย — **คลิกเลือกมัน** ก่อนทำงานต่อทุกครั้ง (เพื่อบอกว่าเรากำลังทำงานในฐานข้อมูลนี้)
